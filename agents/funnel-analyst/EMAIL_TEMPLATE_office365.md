# Office 365 access-request email template

Only needed **if** a future version of the agent must connect to Office 365
(Teams / Outlook / personal SharePoint). The core funnel agent does **not**
require this. Send to **helpdesk@vng.com.vn** (first-response SLA: 8 hours).
Tip from the guideline: if you only need Outlook/Calendar, you can simulate with
a personal Google (Gmail/Calendar) account — instant, no approval needed, and
still valid for the demo.
